I work at the intersection of security and infrastructure — building systems that are not only functional but resilient. My background spans detection engineering, cloud security, and software development, with a focus on practical, impactful work.

## What I build

Security tools, infrastructure automation, and the occasional creative project. I believe the best security solutions are the ones that make the system better for everyone, not just the security team.

My work spans cloud security architectures, detection engineering pipelines, CTF platforms, and homelab infrastructure. Each project is an opportunity to push both my understanding and the capabilities of the tools I use.

## Approach

I'm driven by curiosity and pragmatism. Security is a systems problem first — understanding how things connect, where they break, and how to make them more robust without sacrificing usability.

I write code that outlives the immediate problem. Clean interfaces, clear boundaries, and minimal surprises.

## Beyond the terminal

When I'm not building or breaking things, you'll find me traveling, gaming, or diving into a new homelab project. I believe the best engineers are the ones who explore beyond their immediate discipline.
